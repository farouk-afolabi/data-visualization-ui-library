Accordion & Dropdown

Accordion

Light Theme
What is an accordion?

An accordion is a vertically stacked list of headers that can be clicked to reveal or hide associated content. It's commonly used to break down complex content into digestible sections, making information easier to find and understand.

When to use an accordion menu?
  • To organize related content into collapsible sections
  • When space is limited and content needs to be condensed
  • For FAQ sections or detailed documentation
  • When users need to focus on specific content while having easy access to other sections
What is the best practices?
  • Keep headers clear and concise
  • Use consistent styling for active and inactive states
  • Ensure adequate spacing between sections
  • Include visual indicators (like arrows or icons) to show expand/collapse state

Dropdown Menu

Light Theme
What is a dropdown menu?

A dropdown menu is similar to an accordion but allows multiple sections to remain open simultaneously. It's useful when users need to compare information across different sections.

When to use dropdown menus?
  • When users need to view multiple sections at once
  • For comparing information across different categories
  • When content sections are independent of each other
  • In situations where viewing multiple options simultaneously is beneficial
Best practices for dropdowns
  • Use clear and descriptive headers
  • Maintain consistent styling
  • Ensure adequate spacing between items
  • Use visual indicators for open/closed states
